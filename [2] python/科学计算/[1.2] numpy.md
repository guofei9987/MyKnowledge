# 多维取数规则
```
import numpy as np
a=np.arange(3*4*5).reshape(3,4,5)

lidx=[[0],[1],[2]]

a[lidx]
a[np.array(lidx)]
#结果不一样
```

```python
a.argmax()#返回的是平滑后的位置

#如果想要返回idx，需要用np.unravel_index()转换以下：
idx=np.unravel_index(a.argmax(),a.shape)
a[idx]

#或者将a平滑化
a.ravel()[a.argmax()]
```
