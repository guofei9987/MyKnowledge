## 问题1：正整数分化问题：

正整数n等于若干个正整数的和，这些正整数最大为m，问共有多少种算式。
伪代码：
```
int P(n,m)
{
if(m==1) return 1;
if(m>n) return P(n,n);
if(m==n) return 1+P(n,m-1);
return P(n,m-1)+P(n-m)
}
```
结论：几乎任何复杂递归算式，都能够转化为递归问题。
